# Archivos BPMN

Esta carpeta contiene 30 modelos BPMN base del sistema Calzatura Vilchez.

Uso sugerido:

1. Abrir Bizagi Modeler.
2. Usar la opcion de importar/abrir archivo BPMN si esta disponible.
3. Seleccionar el archivo `.bpmn` requerido.
4. Ajustar carriles, responsables y detalles visuales segun la rubrica del docente.

Los modelos son una base estructurada: inicio, actividades principales, flujos y fin. Para una entrega final en Bizagi, se recomienda agregar piscinas/carriles por actor cuando el docente lo solicite.
