# Instrumento SUS — Calzatura Vilchez (Brooke, 10 ítems)

**Versión:** 1.0  
**Uso:** Post-tarea, después del recorrido guiado en la sesión de usabilidad.  
**Escala:** 1 = Totalmente en desacuerdo … 5 = Totalmente de acuerdo.

> No rellenar con datos de ejemplo en el repositorio. Cada participante completa una copia en papel o formulario; las puntuaciones se consolidan en el acta de sesión.

---

## Instrucciones al participante

Piense en la **aplicación web Calzatura Vilchez** que acaba de usar. Indique su grado de acuerdo con cada afirmación.

| # | Ítem (español adaptado) | 1 | 2 | 3 | 4 | 5 |
|---|-------------------------|---|---|---|---|---|
| 1 | Creo que me gustaría usar este sistema con frecuencia | | | | | |
| 2 | Encontré el sistema innecesariamente complejo | | | | | |
| 3 | Pensé que el sistema era fácil de usar | | | | | |
| 4 | Creo que necesitaría la ayuda de un técnico para poder usarlo | | | | | |
| 5 | Encontré que las funciones del sistema estaban bien integradas | | | | | |
| 6 | Pensé que había demasiada inconsistencia en el sistema | | | | | |
| 7 | Imagino que la mayoría aprendería a usarlo muy rápidamente | | | | | |
| 8 | Encontré el sistema muy engorroso de usar | | | | | |
| 9 | Me sentí muy seguro/a usando el sistema | | | | | |
| 10 | Necesité aprender muchas cosas antes de poder usarlo bien | | | | | |

---

## Cálculo de puntuación (aplicar por participante)

| Ítem | Tipo | Fórmula score (0–4) |
|------|------|---------------------|
| 1, 3, 5, 7, 9 | Impar | `respuesta − 1` |
| 2, 4, 6, 8, 10 | Par | `5 − respuesta` |

**SUS individual = (suma de 10 scores / 25) × 100**

**SUS sesión = media aritmética de los SUS individuales**

### Umbrales de interpretación (literatura)

| Rango SUS | Interpretación |
|-----------|----------------|
| < 50 | Deficiente |
| 50–68 | Por debajo del promedio |
| 68–80 | Por encima del promedio |
| > 80 | Excelente |

**Umbral de aceptación del proyecto (RNF-USA-01): ≥ 70**

## Tareas sugeridas antes del cuestionario

1. Abrir la tienda y localizar un producto en el catálogo.  
2. Añadir al carrito y revisar cantidades.  
3. Iniciar checkout (entrega o recojo en tienda según disponibilidad en demo).  
4. *(Opcional admin)* Solo si el participante es perfil administrador: abrir panel y localizar listado de pedidos.

Registrar tiempos y errores en el acta de sesión (ISO 9241-11 — eficacia/eficiencia).
